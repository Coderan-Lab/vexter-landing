## About

Marketing website for **Vexter**, a SaaS platform that automates incident response in security operations centers (SOC) through AI-powered voice calls.

## Tech Stack

| Technology | Purpose |
|:---|:---|
| **Next.js 14** | Framework (App Router) |
| **TypeScript** | Type safety |
| **Tailwind CSS** | Styling |
| **Framer Motion** | Animations |

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
